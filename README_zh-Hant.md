
# On-chain Card Game

## 安裝依賴

```bash
npm install
```

## 編譯合約

```bash
npx hardhat compile
```

## 部署合約到本地 hardhat 節點

```bash
npx hardhat node
```

另開一個終端，運行部署指令：

```bash
npx hardhat run scripts/deploy.js --network localhost
```

## 遊玩指令行遊戲

```bash
npx hardhat run scripts/play.js --network localhost
```
